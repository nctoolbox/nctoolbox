<p><h5>Summary:</h5><small>nctoolbox is a <a href="http://www.mathworks.com/">Matlab</a> toolbox that provides read-only access to <a href="http://www.unidata.ucar.edu/software/netcdf-java/CDM/index.html">common data model</a> datasets. Under the hood, nctoolbox uses <a href="http://www.unidata.ucar.edu/software/netcdf-java/">NetCDF-Java</a> as the data access layer. This allows nctoolbox to access <a href="http://www.unidata.ucar.edu/software/netcdf/">NetCDF</a>, <a href="http://opendap.org/">OPeNDAP</a>, <a href="http://www.hdfgroup.org/HDF5/">HDF5</a>, GRIB, GRIB2, HDF4 and many (15+) other file formats and services using the same API. It works with Matlab 2008a and later.</small></p>
